# Повышение результативности ПМСП — публичный сайт проекта

Публичный сайт проекта ФГБУ «ЦНИИОИЗ» Минздрава России «Повышение результативности первичной медико-санитарной помощи».

Период отчёта: 2024–2026.  
Язык сайта: русский.  
Целевая аудитория: широкая общественность, партнёры, участники проекта, представители органов власти.

## Структура

- `index.html` — главная страница публичного отчёта;
- `part1.html` — страница разделов проекта и 8 направлений образа результата ПМСП;
- `best-practices.html` — страница лучших практик и банка решений;
- `style.css` — общий файл стилей;
- `assets/` — изображения, иконки и материалы;
- `assets/materials/` — документы, презентации, методические материалы и другие файлы, которые будут добавлены позже.

## Публикация через GitHub Pages

1. Создать репозиторий на GitHub.
2. Загрузить файлы сайта в корень репозитория.
3. В настройках репозитория открыть **Settings → Pages**.
4. Выбрать источник публикации: **Deploy from a branch**.
5. Выбрать ветку `main`, папку `/root`.
6. Сохранить настройки и дождаться публикации сайта.
